# 台湾省数据可视化大屏

基于 Vue 3 + Vite + ECharts + TailwindCSS 的数据可视化大屏。

以台湾省22个县市的模拟统计数据为基础，展示 GDP、综合发展指数、数据中心网络拓扑、地图分布、政务云资源、异常处理、关键词云等多维度数据。

## 启动方式

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5173

## 改了什么

- 八个可视化图表：横向柱状图、雷达图、关系图、地图、数据总览、纵向柱状图、环形图、词云
- 年份筛选（2020-2025），切换年份数据实时更新
- 导出当前年份 JSON 数据
- 台湾省 GeoJSON 地图，支持缩放与拖拽
- 自定义 LXGW 字体与 Electronic 数字字体
- 全局统一的绿色系（#00b894）设计风格
